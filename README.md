A modern, responsive real estate website for Marci Metzger - The Ridge Realty Group, featuring professional design, enhanced user experience, and comprehensive functionality.

## 🚀 Getting Started

### **Local Development**
1. **Clone/Download** the project files
2. **Open** `index.html` in your web browser
3. **Or use a local server**:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (if available)
   npx live-server --port=3000
   ```
4. **Visit** `http://localhost:8000` or `http://localhost:3000`

### **Production Deployment**
1. **Upload** all project files to your hosting platform
2. **Ensure** `index.html` is in the root directory
3. **Test** the website on different devices
4. **Update** any absolute paths if needed
